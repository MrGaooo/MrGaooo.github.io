营销云（B端通用）组件设计—表

Feb 27, 2020

营销云平台项目的底层基础组件使用Ant Design ，是一个非常优秀的企业级产品的设计体系 ，几乎可以满足大多数中后台系统的设计要求，她秉承基于「自然」、「确定性」、「意义感」、「生长性」四大设计价值观，通过模块化解决方案，降低冗余的生产成本，让设计者专注于更好的用户体验。

对于营销云平台来说也能很大程度上满足了我们的使用需求，让我们能够快速高效的搭建系统。

但是，在营销云平台中有一些有特殊功能的组件还是不能够满足，所以需要我们结合Ant Design的基础组件设计来创造属于营销云平台专属的功能组件。

Table 表格

对于营销云平台用户而言，来后台主要目的之一就是对数据进行查阅和操作，而表格做为结构化数据的载体，承载了数据的保存、展示、对比分析、排序、筛选 、搜索等功能。

因此表格的易读性和易操作性设计对于提升营销云平台用户的操作效率有着举足轻重的作用。

营销云平台的表格由内外两部分组成：

  • 内部，由表头、表体、表尾共3 部份组成。
  • 外部，由搜索、筛选、相关命令按钮、分页等构成。

在设计上营销云平台的表格采用最简洁的设计风格，去掉了纵向的分割线

功能

营销云平台的表格主要分为查看和编辑两个状态

查看状态

查看状态列表分为有编辑功能、无编辑功能两种

无编辑功能（eg：最近浏览、搜索结果）

  1. 排序（单行排序、组合排序）、清除排序
  2. 选择、全选
  3. 删除
  4. 隐藏、显示全部
  5. 拖拽列宽
  6. 操作：查看详情、更多操作
  7. 帮助

有编辑功能（eg：详情页子表）

  1. 排序（单行排序、组合排序）、清除排序
  2. 隐藏、显示全部
  3. 拖拽列宽
  4. 操作：查看详情、更多操作
  5. 帮助

​ 此类列表选择和删除功能仅在编辑状态下可用

编辑状态

当进入编辑状态时列表顺序恢复初始顺序且不可排序（单行排序、组合排序）

  1. 选择、全选
  2. 删除
  3. 拖拽排序
  4. 隐藏、显示全部
  5. 拖拽列宽
  6. 保存、取消
  7. 操作：编辑、更多操作
  8. 帮助
